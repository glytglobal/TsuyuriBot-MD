### `PARA ACTIVAR EN TERMUX`
```bash
$ pkg upgrade && pkg update
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/glytglobal/TsuyuriBot-MD
$ cd TsuyuriBot-MD
$ npm i 
```
```bash
$ node .
```
